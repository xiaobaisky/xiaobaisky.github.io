陈氏太极拳之老架二路（炮锤）拳谱

2023-12-21T12:25:45.000Z5.1k17 分钟China

此拳系马虹支系，于今年在南湖公园从郭庆法老师处习得，此拳，在一路基础上，增加了好多发力动作，柔中有刚，折叠往复，阴阳相济，是一路的进阶，人人可习，人人可练，一可强身健体，二可防身自卫。再此作记录备忘用，遂作此文。

第1式  预备式

虚领顶劲，头要正，两眼平视前方，下颌微收，两臂自然下垂，中指贴裤缝，挺胸收腹，提气提肛，两腿自然微曲，两脚与肩同宽，平行抓地，两肩微微内合。

第2式  金刚捣碓

  1. 踏掌根，自右向左提，到右胸前，按；
  2. 同时踏掌根，转体摆脚，捋向东北方向75度，立身中正，重心在左，两手距离一小臂远，左掌在胸前中线，右臂掤圆；
  3. 裆走下弧倒重心，提左膝，肘膝相合；
  4. 西北45度擦脚，走捋拌；
  5. 重心在右，左脚尖摆向正北，前移重心成左式弓蹬步，两手臂成三吊腕；
  6. 重心后移（欲前先后），三点同时到位；
  7. 下沉合，再下沉握拳，提，砸。

第3式  懒扎衣

金刚捣碓定式

  1. 提掌拳到右胸前，再下旋划圆到右胸前变十字手；
  2. 前掤，命门穴后撑；
  3. 重心右移开，右掌左穿向上向右划弧，左掌向左下按；
  4. 插左掌提右膝，肘膝相合，掌根相对；
  5. 引，东北方向右前一脚的三分之二处插脚，同时右掌自右向左前方引；
  6. 靠，左掌搭右肘；
  7. 重心右移，扣左脚，右掌踏掌根掤；
  8. 挒（胸前略走上弧自左向右），左手下沉放气海穴，指尖向里，转腰，肘膝相合，下踏外辗。

第4式  六封四闭

懒扎衣定式

  1. 掤，逆时针转动丹田加掤劲（左掌随丹田转动，右掌在右膝前转动，掌心向里加掤劲）；
  2. 引（右掌自上右下引至左胸前,向下向外引）；
  3. 靠，重心右移（肩背靠）；
  4. 右掌搭左掌成阴阳手（右上左下）；
  5. 提，胸腰折叠转体挤（重心右移，右掌外左掌内，手背相搭）；
  6. 吊，重心左移，右手自上往下划圆，至胸前中线上托，左掌上吊，与太阳穴同高；
  7. 手开肘合；
  8. 手合肘开，双手至耳朵两侧，指尖靠近耳朵；
  9. 后弧按，向右胯下方按，同时左脚划弧跟进呈扇形。

第5式  单鞭

六封四闭定式，以左脚掌为轴

  1. 按，穿掌锁喉（左掌斜向前上至对方喉部，右掌随左掌至左肘，掌心向上）；
  2. 扣掌合肘，（右掌旋腕成勾至左肘，左掌托右肘）；
  3. 给肘勾手（右肘右击后伸右臂勾手），左手下沉至气海；
  4. 甩头提膝插脚（头向左甩，提左膝向左后三分之二插脚），勾手竖直；
  5. 靠（移重心，左肩左挤，左脚尖同时摆一下）；
  6. 略扣右脚，倒重心，左掌自丹田右胸右肩窝穿，踏掌根，转腰同时左掌略走上弧至左膝上，右勾随转动至右膝上，勾手变圆虚握；
  7. 还原，肘膝相合，下塌外辗。

第6式 搬拦肘

单鞭定式

  1. 双掌略沉，走上弧自右划到左侧握拳（阴阳拳）再向右捋，同时双足向上跳起，双脚掌脚跟同时落地，左拳在腹前中线，拳心向上，右拳在右胯外侧，拳心向下；
  2. 正西方向上左脚跟右脚，同时腰用劲双拳撞向正西，左拳在左胯外侧，拳心向上，右拳在腹前中线，拳心向下；
  3. 双拳变掌，走上弧自左划到右侧握拳（阴阳拳）再向左捋，同时双足向上跳起，双脚掌脚跟同时落地，右拳在腹前中线，拳心向上，左拳在左胯外侧，拳心向下；
  4. 正东方向上右脚跟左脚，同时腰用劲双拳撞向正东，右拳在右胯外侧，拳心向上，左拳在腹前中线，拳心向下。

第7式  跃步护心捶

搬拦肘定式

  1. 双脚后跟往右后摆，掤左臂穿右拳；
  2. 双拳前掤下捋，左拳在腹前中线，拳心向上，右拳在右胯外侧，拳心向下；
  3. 翻身跃步砸过来（方向西南正马步），右拳在上，左拳在下，拳眼相对，和于胸前；
  4. 双顺缠开（距离20-30公分)，双逆缠再开，转体进肘；
  5. 合；
  6. 转体做护心捶（左拳自上向下经右胸前划弧至胸前中线略靠下，右拳自上向下经胸前划弧旋至颈侧肩上，由腰带动砸至胸前中线左拳之上，双臂掤圆，拳心向内，马步。

第8式  跃步斜行

跃步护心捶定式

  1. 弹击右掌（右掌背右斜上方向击到对方脸部），左手半折腕贴左胸；
  2. 插左掌（腹前中线，掌心向外），右掌贴左肩窝；
  3. 扣左脚，掌心相错开，右脚划后弧跟；
  4. 左掌自上划向右下，右掌向上划，提右膝，向前跃步，左手自右胸前划弧向下搂过膝勾手上提，同时右手自右耳前右肩窝位置自下向上按，双手与胸同高；
  5. 踏掌根（右掌），转腰，挒（胸前自左向右至右膝上方，左勾手，手背向前随腰转），到中线；
  6. 再踏掌根，还原，下塌，两臂掤圆，左手勾手变圆，立身中正，方向正西，腿西南东北方向成左式弓蹬步。

第9式  煞腰压肘拳

跃步斜行定式

  1. 两掌走上弧自左向右到右胸前；
  2. 握拳向左捋，右拳在腹前中线，拳心向上，左拳在左胯外侧，拳心向下；
  3. 跳起转体砸（腿的方向西北东南），右拳砸到右胯外侧，拳心向上，左拳打到左太阳穴高度，拳面斜朝上，拳心斜向内。

第10式  井缆直入

煞腰压肘拳定式

  1. 转腰上身转向正北，左右拳同时下插，右拳在腹前中线，左拳在左胯外侧，双臂三吊腕，拳心向后；
  2. 转体摆脚，右掌前掤，与额同高，掌心向外，左掌贴左肩窝；
  3. 东南方向上左脚，前脚掌着地，走左肘自上向下压肘，左掌下探左胯外侧穿掌，掌心向外，右掌掌心向上收到左胸前。

第11式  风扫梅花

井缆直入定式

  1. 掤（右掌向前掤，左掌在左胯下按），双脚后跟向左后摆；
  2. 倒重心，扫右腿转体，双腿西北东南方向，左掌上托，右掌下按。

第12式  金刚捣碓（同一路）

风扫梅花定式

  1. 腰带动同时翻掌；
  2. 三点同时到位；
  3. 下沉合，再下沉握拳，提，砸。

第13式  披身捶（同一路）

金刚捣碓定式

  1. 开（腰和双手同时水平逆时针转，双掌掌心向上收于腰侧）；
  2. 右脚向右跨半步双掌划圆弧至胸前成十字手左掌在外成正马步；
  3. 滑步进肘（向右滑步冲右肘，十字手变拳，右拳和左腕）；
  4. 顺缠开（两拳心向内）；
  5. 逆缠再开（两拳心向外）；
  6. 转体给肘（拳开肘合，两拳心斜相对与太阳穴同高，两肘收于胸前）。

背折靠:

  1. 合（两拳心向内）；

  2. 转体採（左肘向下竖于左胸前，拳心向内，右拳至右膝外侧，拳心向上）；

  3. 横打（左拳收于腰际，右拳上提划弧至太阳穴高度自右向左手背向前至右胸前）；

  4. 沉肘转腕给肘（右拳与太阳穴同高，同时左拳向内扣腰带动左肘向前），右偏马步，左肘左膝相合。

第14式  撇身捶

披身捶背折靠定式

  1. 开合（双拳逆缠开，小碎步向西跨，同时顺缠合，右拳合到左肘处，左拳拳心向上右拳拳心贴左肘弯）；
  2. 继续开合（同1）；
  3. 跳起走合开（半转身双腿起跳，同时左拳合到右腋下，右拳到左肩窝，双足同时落地方向西南东北，双拳撇开，左拳西南，拳心向上，不超肩，右拳在右跨上方，拳心向下），双腿呈左式弓蹬步。

第15式 斩手

撇身捶定式

  1. 双拳顺时针前掤，下捋（左拳在腹前中线，拳心朝上，右拳在右胯外侧，拳心朝下）；
  2. 摆左脚，重心前移，双拳上打（左拳变掌掌心与右拳拳心相对，与脸同高，与头同宽）；
  3. 上右脚垫步（双足平行抓地，相距20到30公分），双膝微曲，同时拳掌走开合于胸前（右拳面贴左掌心）。

第16式  翻花舞袖

斩手定式

  1. 穿掌（左掌穿向西南，右拳合于左肘）；
  2. 合肘（收左掌合到右肘）；
  3. 挒（双手成虎爪，左爪上挒到左胸，右爪下挒到右腹，双爪心朝里）；
  4. 跳起转体（360度）双臂轮劈（双腿方向西北东南，成右式弓蹬步）。

第17式  掩手肱捶

翻花舞袖定式

  1. 扣左脚提膝上打；
  2. 垫步下打；
  3. 西南方向擦脚，靠；
  4. 摆拳，合肘；
  5. 拉弓式开，再开；
  6. 扭腰送胯转体将右拳合力击出，左拳成老虎爪贴左胸。

第18式  飞步拗鸾肘

掩手肱捶定式

  1. 掤，转动丹田加掤劲（左掌随丹田转动，右掌在右膝前转动，掌心向里加掤劲）；
  2. 提，左肘左膝相合；
  3. 击拳（击右拳）跃步转体；
  4. 开（左臂开），合（合向右肘），右脚掌点地，右脚跟抬起。

第19式  运手

飞步拗鸾肘定式，双掌走上弧，收脚收手，加掤劲，略踏左脚，伸右脚，重心在左。运手:

  1. 穿右掌，到左肩窝，踏掌根，上左脚（到右后方），砍左掌，左掌在腹前中线，指尖斜向下，右臂膨圆，右掌略高于肩，掌心向外；
  2. 穿左掌，踏掌根，砍右掌，同时向右侧伸右脚；
  3. 继续穿右掌，踏掌根，砍左掌，穿左掌，踏掌根，砍右掌。

第20式  高探马

运手定式

  1. 双掌打向正东；
  2. 击左掌收右掌（右掌收于气海，指尖向里）；
  3. 击右掌收左掌（左掌收于腰迹）；
  4. 击左掌提右膝转体落步向右掤。

第21式  运手

高探马定式，落步加掤劲，略踏右脚，伸左脚，重心在右。运手:

  1. 穿左掌，到右肩窝，踏掌根，上右脚（到左后方），砍右掌，右掌在腹前中线，指尖斜向下，左臂膨圆，左掌略高于肩，掌心向外；
  2. 穿右掌，踏掌根，砍左掌，同时向左侧伸左脚；
  3. 继续穿左掌，踏掌根，砍右掌，穿右掌，踏掌根，砍左掌。

第22式  高探马

运手定式

  1. 摆左脚,右脚划后弧跟，左臂掤圆，左掌略高于肩，右掌在腹前中线，掌心向左，指尖斜向下；
  2. 正南方向擦脚，左掌搭右肘，靠，踏，掤，走肩走肘走手，还原（身体方向正东）；
  3. 双掌掌心向上合到胸前，然后开、合（象倒卷肱，双开双合，左掌在前，右掌在右肩颈，双掌一线，双肘一线）；
  4. 左掌踏掌根拽对方手，击右掌转体（腰带动左脚向左后扫，右掌击对方太阳穴，左掌收到气海，指尖向内），左脚划后弧到右脚侧后方，前脚掌着地，脚底扇形。方向东北。

第23式  连环炮（一）

  1. 双腿滑步后撤，双腕自下而上前掤，两指尖向内；
  2. 吊；
  3. 提膝开；
  4. 落脚合；
  5. 上步推。

第24式  连环炮（二）

  1. 双腿滑步后撤，双腕自下而上前掤，两指尖向内；
  2. 吊；
  3. 提膝开；
  4. 落脚合；
  5. 上步推。

第25式  连环炮（三）

  1. 双腿滑步后撤，双腕自下而上前掤，两指尖向内；
  2. 吊；
  3. 提膝开；
  4. 落脚合；
  5. 上步推。

第26式  倒骑麟

  1. 转向西北合；
  2. 扣左脚，踏掌根转体摆脚；
  3. 蹬 ，转体盖掌，右手收于气海穴。

第27式  白蛇吐信（一）

上步穿掌。

第28式  白蛇吐信（二）

  1. 掤右臂，倒重心到右腿；
  2. 提左膝盖掌，右手收于气海穴；
  3. 上步穿掌。

第29式  白蛇吐信（三）

  1. 掤右臂，倒重心到右腿；
  2. 提左膝盖掌，右手收于气海穴；
  3. 上步穿掌。

第30式  海底翻花

  1. 双掌握拳，扣左脚，倒重心到左腿；
  2. 转体提右膝，左臂掤圆，左拳拳心向里与太阳穴同高，右拳扳砸在右膝外侧，拳心向上。

第31式  掩手肱捶（二）

  1. 上打；
  2. 垫步下打；
  3. 东北方向擦脚；
  4. 摆拳，合肘；
  5. 开，再开；
  6. 打。

第32式  转身六合

  1. 正东跃步合（两拳心均向外）；
  2. 拳心向下开与膝同宽；
  3. 腹前合，翻拳再开（两拳心向上）；
  4. 扣左脚，转身提右膝，双腕合于腹前，左拳在外，右拳在内，两拳心向内。

第33式  左裹鞭炮（一）

  1. 落右脚，向西上左脚成正马步；
  2. 裹，开。

第34式  左裹鞭炮（二）

  1. 右脚在左脚前向西上步，左脚向西开步成正马步；
  2. 裹，开。

第35式  右裹鞭炮（一）

  1. 左肘胸前中线採，右拳扳砸在右胯外侧；
  2. 扣右脚，扫左脚转体（面朝正南）合（双腕合于腹前，正马步，左拳在外，右拳在内，两拳心向内）；
  3. 裹，开。

第36式  右裹鞭炮（二）

  1. 向东跃步合；
  2. 裹，开。

第37式  兽头式

  1. 两脚后跟向左后摆，同时掤右拳穿左拳；
  2. 左拳在胸前中线自下而上向外扳击，右拳收回护胸，双臂掤圆，同时右脚划内弧开，重心靠前；
  3. 右拳在胸前中线自下而上向外扳击，左拳收回护胸，双臂掤圆，同时左脚划内弧开，重心靠前。

第38式  劈架子

  1. 弹击右掌，左掌腕贴左胸；
  2. 掌背相错开，右脚划后弧跟；
  3. 上步合（左掌在两腿中线掌心向上，右掌贴左肩窝掌心向外）；
  4. 跳起合开。

第39式  翻花舞袖

  1. 双掌前掤，下捋；
  2. 转体轮劈，腿的方向西北东南。

第40式  掩手肱拳（三）

  1. 扣左脚提右膝上打；
  2. 垫步下打；
  3. 西南方向擦脚；
  4. 摆拳，合肘；
  5. 开，再开；
  6. 打。

第41式  伏虎

  1. 两拳双顺缠开，双逆缠再开；
  2. 转身进肘；
  3. 转体左拳转到左膝上方，拳心向上，右拳与右太阳穴同高，拳心向里。

第42式  抹眉红

  1. 西南方向滑步继续下沉，右拳变掌贴右肩窝处；
  2. 提左膝，跃步转体，左拳收于腰际，右掌掌心向外虎口朝下，借转体之势自右向左抹（与眼同高），抹向东北方向。

第43式 右黄龙三搅水

  1. 撤左腿掤右掌；
  2. 收脚收手引，右手引到腹前中线；
  3. 东南方向撤右脚，右手继续向前引，到左胸前，掌心朝上；
  4. 踏掌根左腿转体扫，同时左肘前顶，眼看肘膝方向；
  5. 还原（右掌在右膝外侧上方，掌心向前）；

重复1-5两遍。

第44式 左黄龙三搅水

  1. 双掌前掤，下捋；
  2. 跳起转体轮劈，腿的方向东北西南；
  3. 撤右腿掤左掌；
  4. 收脚收手引，左手引到腹前中线；
  5. 东北方向撤左脚，左手继续向前引，到右胸前，掌心朝上；
  6. 踏掌根右腿转体扫，同时右肘前顶，眼看肘膝方向；
  7. 还原（左掌在左膝外侧上方，掌心向前）；

重复3-7两遍。

第45式  左蹬脚

  1. 收（撤左脚，双掌收于腹前，双腕相搭，左掌在上，右掌在下，掌心朝上）；
  2. 向西跨，马步开，两臂掤圆与两膝同宽不超肩；
  3. 收脚收手，扣，下沉，翻掌握拳，再翻拳蹬左脚。

第46式  右蹬脚

  1. 左脚下落到东北方向，脚后跟着地，双拳腕相搭，左拳在上，右拳在下，拳心朝上；
  2. 扣，翻拳蹬右脚。

第47式  海底翻花

东南方向

第48式  掩手肱捶（四）

东南方向

第49式  扫堂腿（转胫炮）

  1. 重心右移掤；
  2. 合肘（右拳合左肘）；
  3. 扣左脚，收右脚震右脚，西北方向撤右脚；
  4. 倒重心到右腿，顺时针转体扫左腿450度，面向西北。

第50式  掩手肱捶（五）

  1. 西北方向，提膝上打；
  2. 垫步下打；
  3. 西南方向擦脚，靠，摆拳，合肘，开，再开，打。

第51式  左冲

  1. 合（双拳合于胸前，双拳心向内）；
  2. 胸腰折叠，跳起向左转体（面向西南）砸（双拳合于胸前）；
  3. 胸腰折叠弹（双足跳起，双拳向前弹击）。

第52式  右冲

  1. 胸腰折叠，跳起向右转体（面向西北）砸（双拳合于胸前，双拳心向内）；
  2. 胸腰折叠弹（双足跳起，双拳向前弹击）。

第53式  倒插

  1. 拳开肘合；
  2. 右脚上步栽拳（右脚上小半步脚掌着地，右拳下击栽拳三吊腕，左拳贴右肩窝，两拳心向外）。

第54式  海底翻花

西北方向

第55式  掩手肱捶（六）

西北方向

第56式  夺二肱（一）

  1. 重心后移两臂掤圆；
  2. 扣左脚倒重心到左腿；
  3. 向右转体合（面向东北，右拳合左肘），右脚划后弧跟；
  4. 开（双臂逆缠开），提膝（右膝）；
  5. 垫步合肘；
  6. 上步（右脚）弹击右拳，左手虎爪收于腰际。

第57式  夺二肱（二）

  1. 开（双臂平展，拳心向下，向西蹬左脚）；
  2. 蹬，合（左脚向东蹬，合肘），落；
  3. 提膝开；
  4. 垫步合；
  5. 上步打。

第58式   连珠炮

击左拳右拳护肘，击右拳左拳护肘，再击左拳右拳护肘，三拳打击同一方位。

第59式  左玉女穿梭

  1. 掤右拳提右膝，右拳收于腰迹；
  2. 跃步击左拳转体（上步击左拳（先收再击），跳起向右转体（面向西北），落地正马步，双拳置于胸前，拳面相对，拳心向内）。

第60式  回头当门炮

胸腰折叠跳起撞拳。

第61式  右玉女穿梭

  1. 掤右拳提左膝，右拳收于腰迹；
  2. 跃步击右拳转体（上步击右拳，跳起向左转体（面向东北），落地正马步，双拳置于胸前，拳面相对，拳心向内）。

第62式  回头当门炮

胸腰折叠跳起撞拳。

第63式  撇身捶

  1. 跳起左转身（面向西北）合（左拳腹前中线前引，拳心向上，右拳贴左肩窝）；
  2. 跳起合开，两腿西南东北方向，左拳不超肩，拳心向上，右拳在右胯外侧，拳心向下。

第64式  拗鸾肘

  1. 双拳前掤，变掌下捋；
  2. 摆左脚，重心前移，掤击左掌，右掌变拳收于腰际，拳心向上；
  3. 开，合垫（上右脚震脚给肘，左掌贴右小臂）；
  4. 掌拳相握，自右向上划过头顶到左侧，同时提右膝，落步，左掌托右拳背向右给肘击打。

第65式  顺鸾肘

掌拳相握，自右向上划过头顶，同时提右膝，落步，双拳成虎爪贴胸，双肘向身后给肘击打。

第66式  穿心肘

掌拳相握，自右向上划过头顶到左侧，同时提右膝，落步，左掌托右小臂向右给肘击打。

第67式  窝里炮（又称窝底炮）

  1. 双腿向后小滑步，右拳自内向外，自下向上弹，左掌托右腕；
  2. 收脚收手（收右脚，右拳引，左掌变拳收于腰际）；
  3. 撤右脚（撤向西北）右拳继续引；
  4. 重心右移，右拳拳背斜上击打。

第68式  井缆直入

  1. 摆右脚，重心前移，收右拳变掌前掤，与额同高，掌心向外，左拳变掌贴左肩窝；
  2. 上左脚，走左肘自上向下压肘，左掌下探左胯外侧穿掌，掌心向外，右掌掌心向上收到左肩窝。

第69式  风扫梅花

  1. 双脚跟向左后摆，掤右掌，切左掌；
  2. 倒重心，扫右腿转体，双腿东南西北方向，左掌上托，右掌下按。

第70式  金刚捣碓

  1. 腰带动同时翻掌；
  2. 三点同时到位；
  3. 下沉合，再下沉握拳，提，砸。

第71式  收式

  1. 开（腰和双手同时水平逆时针转，双掌掌心向上收于腰侧）；
  2. 翻掌下按，身体站直，双臂自然下垂，指尖贴裤缝。抱拳行礼！
