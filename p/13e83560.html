太极陈氏

陈氏太极拳之老架一路拳谱

此拳系马虹支系，于今年在南湖公园从郭庆法老师处习得，此拳，柔中有刚，折叠往复，阴阳相济，是二路的基础，人人可习，人人可练，一可强身健体，二可防身自卫。再此作记录备忘用，遂作此文。

深抠拳谱细分析

弄清拳理找规律

钻研拳法懂劲道

持之以恒见功底

主宰与腰、基于脚、行于手,意领形，腰走劲，虚灵顶劲，沉肩坠肘，松胯圆裆，抓拿式。

第1式 预备式

虚领顶劲，头要正，两眼平视前方，下颚微收，两臂自然下垂，中指贴裤逢，挺胸收腹，提气提肛，两腿自然微曲，两脚与肩同宽，平行抓地，两肩微微内合。

第2式 金刚捣堆

  1. 踏掌根，自右向左提，到右胸前；
  2. 同时踏掌跟，转体摆脚，捋向西南75度，立身中正，重心在左，两手距离一小臂远，左掌在前胸中线，右臂棚圆；
  3. 裆走下弧倒重心，提左膝，肘膝相合；
  4. 东南45度擦脚，走捋拌；
  5. 重心在右，左脚尖摆向正南，前移重心成左式弓蹬步，两手臂成三吊腕；
  6. 重心后移（欲前先后），三点同时到位；
  7. 下沉合，再下沉握拳，提，砸。

第3式 懒扎衣

金刚捣堆定式

  1. 提掌拳到右胸前，再下旋划圆到右胸前变十字手；
  2. 前棚，命门穴后撑；
  3. 重心右移开，右掌左穿向上向右划弧到头前中线，左掌向下按到左胯外侧；
  4. 插左掌提右膝，肘膝相合，掌根相对；
  5. 引，西南方向右前一脚的三分之二处插脚，同时右掌自右向左前方引；
  6. 靠，左掌搭右肘；
  7. 重心右移，扣左脚，右掌踏掌根棚；
  8. 挒（胸前略走上弧自左向右），左手下沉放气海穴，指尖向里，转腰，肘膝相合，下踏外展。

第4式 六封四闭

懒扎衣定式

  1. 棚，逆时针转动丹田加棚劲（左拳随丹田转动，右掌在右膝前转动，掌心向里加棚劲）；
  2. 引，（右掌自上右下引至左胸前，向下向外引）；
  3. 靠，重心右移（肩背靠）；
  4. 右掌搭左掌成阴阳手（右上左下）；
  5. 提，胸腰折叠转体挤（重心右移，右掌外左掌内，手背相搭）；
  6. 吊，重心左移，右手自上往下划圆，至胸前中线上托。左掌上吊，与太阳穴同高；
  7. 手开肘合；
  8. 手合肘开，双手至耳朵两侧，指尖靠近耳朵；
  9. 后弧按，向右胯下方按，同时左脚划弧跟进呈扇形。

第5式 第一单鞭

六封四闭定式

以左脚掌为轴，

  1. 穿掌锁喉（左掌斜向前上至对方喉部，右掌随左掌至左肘，掌心向上）；
  2. 扣掌合肘（右掌旋腕成勾至左肘，左掌托右肘）；
  3. 给肘勾手（右肘右击后伸右臂勾手），左手下沉至气海；
  4. 甩头提膝插脚（头向左甩，提左膝向左后三分之二插脚），勾手竖直；
  5. 靠，（移重心，左肩左挤，左脚尖同时摆一下）；
  6. 略扣右脚，倒重心，左掌自丹田右胸右肩窝穿，踏掌根，转腰同时左掌略走上弧至左膝上，右勾随转动至右膝上，勾手变圆虚握；
  7. 还原，肘膝相合，下塌外展。

第6式 第二金刚捣碓

单鞭定式

  1. 向左棚；
  2. 捋，重心右移左掌至胸前中线，两手距离一小臂远；
  3. 再棚；
  4. 往前送，往回大捋；
  5. 倒重心成左式弓蹬步；
  6. 三点同时到位；
  7. 下沉合，再下沉握拳，提，砸。

第7式 白鹤亮翅

第二金刚捣碓定式

  1. 提掌握拳到右胸前；
  2. ，再下旋划圆到右胸前变十字手；
  3. 前棚，命门穴后撑；
  4. 重心右移开，右掌左穿向上向右划弧，左掌向左下按；
  5. 插左掌提右膝，肘膝相合，掌根相对；
  6. 东南方向45度插脚（插脚不要大了），同时右掌自右向左走下弧引；
  7. 靠，左掌搭右肘，重心右移，右掌踏掌根；
  8. 棚挒，开右胸，右掌胸前靠上自左下向右上至太阳穴，左手下沉至左胯掌心向下，转腰，双臂棚圆，两虎口相对，左脚划弧收成扇形；
  9. 还原，肘膝相合，方向东北，立身中正。

第8式 斜行拗步:白鹤亮翅定式

  1. 开右胸，右拳自右上向左打到左胸前；
  2. 重心左移，摆右脚，转体朝东南方向，左掌自左胸前打向正东，拳心向南，右拳下按至右胯掌心向下；
  3. 重心前移，提左膝，肘膝相合；
  4. 走捋拌（左前方东北方向）；
  5. 走搂推，左手自右胸前划弧向下楼过膝勾手上提，同时右手自右耳前右肩窝位置自下向上按，双手与胸同高；
  6. 踏掌根（右掌），转腰，挒（胸前自左向右至右膝上方，左勾手，手背向前随腰转），到中线；
  7. 再踏掌根，还原，下塌，两肩棚圆，左手勾手变圆，立身中正，方向正东，成左式弓蹬步。

第9式 初收

斜行拗步定式

  1. 松胯，两掌收到右腹前，掌心向下，虎口相对；
  2. 向左前上方棚；
  3. 双掌向外划弧自下而上托向前送，右掌靠左肘，双掌心向上；
  4. 腰带着扣掌；
  5. 略扣右脚，双拳抓拿式收提至前胸；
  6. 提左膝双掌下按。

第10式 前蹚拗步

初收定式

  1. 扬指坐腕向右后方捋；
  2. 左前方（东北方向）上步搭手（左脚自前向左外插脚，左手在前右手搭左腕）；
  3. 摆（踏）脚给肘（右肘）；
  4. 提右膝插脚（右前东南方向）；
  5. 靠；
  6. 给肘给手，双臂棚圆，成马步，立身中正，东北方向。

第11式 第二斜行拗步

前蹚拗步定式

  1. 棚（向右棚，右掌在上，左掌在下腹前中线）；
  2. 引，靠；
  3. 重心右移，摆右脚，转体，朝东南方向，左掌自左胸前正东掌心向南，右掌下按至右胯掌心向下；
  4. 重心向前移，提左膝，肘膝相合；
  5. 左前方（东北方向）走捋拌；
  6. 左手自右胸前划弧向下搂过膝勾手上提，同时右手自右耳向下前上按，双手与胸同高，踏右掌，转腰，挒（胸前自左向右至腹前中线，左勾手，手背向前随腰转），再踏掌根，还原，下塌两臂棚圆，左手勾手变圆，立身中正，方向正东，成左式弓蹬步。

第12式 再收

斜行拗步定式

  1. 松胯，两掌收到右腹前，掌心向下，虎口相对；
  2. 向左前上方棚；
  3. 双掌向外划弧自下而上向上托向前送，右掌靠左肘，双掌心向上；
  4. 腰带着扣掌；
  5. 略扣右脚，双掌抓拿式收提至胸前；
  6. 提左膝双掌下按。

第13式 前蹚拗步

再收定式

  1. 扬指坐腕向右后方捋；
  2. 左前方（东北方向）上步搭手（左脚自前向左外插脚，左手在前右上手搭左腕）；
  3. 摆（踏）脚给肘（右肘）；
  4. 提右膝插脚（右前东南方向）；
  5. 靠；
  6. 给肘给手，双臂棚圆，成马步，立身中正，方向东北。

第14式 掩手肱锤

前蹚拗步定式

  1. 开（向外微开）；
  2. 合（双掌划弧抱圆至小腹前）；
  3. 开（右手变掌自内向下向外砸至右膝上，掌心朝上，左手自下从胸. 脸前提至左太阳穴，变抓拿手，掌心斜向内）；
  4. 扣左脚（尽量扣），转体提右膝上打（两臂划弧，打至对方头部，左掌立，右掌面与左掌心相对，双手距离对方头部远）；
  5. 垫步下打（右拳自胸前向下向外崩打，左掌搭右肘），双脚平行15至20公分，东南方向；
  6. 提左膝插脚（左前方，东北方向）；
  7. 靠；
  8. 摆掌（右掌向外摆击打）；
  9. 合肘（右掌合到左肘外）；
  10. 拉弓式开（掌左向前左，掌心向外，右拳至右胯，掌心向下后）；
    1.1. 再开（双划圆，右掌掌心向上收于腰际，左掌掌心向南）；
    1.2. 用三个合力（扭腰. 送胯. 转体）将右拳螺旋打出，同时左掌成虎掌抓收至左胸。

第15式 十字手

掩手肱锤定式

  1. 棚（转动丹田加棚劲）；
  2. 重心右移开（右掌自上向左下划弧上提至太阳穴，左掌下沉至左膝上，两臂棚圆，掌心相对）成右偏马步；
  3. 掌心前移合（右掌划弧自上向左下，左掌搭右小臂）；
  4. 踏掌根扣左脚成十字手，扫（腰带动右腿向右转，方向正南，成正马步）。

第16式 第三金刚捣碓

十字手定式

  1. 棚；
  2. 开（重心在右，右掌与太阳穴同高，左掌左膝前内切掌）；
  3. 继续下沉穿掌（右掌不动，左掌在左膝外侧，掌心向前下穿）；
  4. 提左膝摆右脚上步落脚踩（同时提左掌左腿迈向左前方，落步左手上托右手下按）；
  5. 腰带动反掌；
  6. 重心后移，三点同时到位；
  7. 下沉合，再下沉握拳，提，砸。

第17式 披身锤含背折靠

第三金刚捣碓定式

披身锤

  1. 开（腰和双手同时水平逆时针转，双掌掌心向上收于腰侧）；
  2. 右脚向右跨半步双掌划圆弧至胸前成十字手左掌在外成正马步；
  3. 滑步进肘（向右滑步，冲右肘，十字手变掌，右拳合左腕）；
  4. 顺缠开（两拳心向内）；
  5. 逆缠再开（两拳心向外）；
  6. 转体给肘（拳开肘合，两拳心斜相对与太阳穴同高，两肘收于胸前）。

背折靠

  1. 合（两拳心向内）；
  2. 转体採（左肘向下竖于左胸前，拳心向内，右拳至右膝外侧，拳心向上）；
  3. 横打（左拳收于腰际，右拳上提划弧至太阳穴高度自右向左手背向前至右胸前）；
  4. 沉肘转腕给肘（右拳与太阳穴同高，同时左拳向内扣腰带动左肘向前），右偏马步，左肘左膝相合。

第18式 青龙出水

披身锤含背折靠定式

  1. 左拳螺旋上升右拳螺旋下降（左拳至右下颚，右掌至右膝外侧）；
  2. 横打（左拳收于腰际，右拳上提划弧至太阳穴高度自右向左手背向前至右胸前）；
  3. 撤步弹（右腿后撤左腿跟随后撤，右掌搭在左肘上部，左臂自由下垂自左向右弹，虎口斜向上指尖向前，掌心向内）；
  4. 合（左掌合右肘）；
  5. 上步再打(右脚右前方上步，右掌自左上向右前下方打，右臂棚圆，掌心向下，左掌半握拳收于腰际)。

第19式 双推掌

青龙出水定式

  1. 转动丹田加棚劲；
  2. 引靠（重心在右，用肩背靠）；
  3. 两拳腕相搭（右腕搭左腕）；
  4. 提，胸腰折叠转体挤（重心右移，右拳外左拳内，拳背相搭，两臂棚圆）；
  5. 採，拳背相搭，两臂棚圆往下採；
  6. 重心右移双拳变掌，右掌上，左掌在右肘内侧，双掌心斜向上；
  7. 摆左脚，倒重心到左腿，转体扫，（两手姿势保持不变，右腿自右向左前方扫，右脚脚趾点地）；
  8. 提右膝，开（双掌指尖相对，掌心向上，与腰同宽）；
  9. 落脚合；
  10. 后弧按（左脚后划弧到右脚跟后，成扇形，双掌自两耳后向右前胸按，至胸前中线，注意

双掌按在胸前，不要过低）。

第20式 三换掌

双推掌定式

  1. 按，穿掌锁喉；
  2. 扣掌合肘；
  3. 走肘走手，击右掌（掌与太阳穴同高），左掌护右肋（先走肘后走手）；
  4. 开右胸再击右掌，左掌护右肋不动；
  5. 第二掌，击左掌，右掌护左肋；
  6. 第三拳，击右掌，左掌护右肋。三掌打击同一方位。

第21式 肘底锤

三换掌定式

  1. 右掌从上往下划圆，同时左掌从下往上划圆；
  2. 同时到左胸前，左掌立斜向上左肘向下，右掌变拳托左肘，拳眼向上，拳心向内。

第22式 倒卷肱

肘底锤定式

  1. 穿掌合肘（左掌前穿自左至右，右肘向外与左掌合）；
  2. 左脚上步划内弧开，下踏松胯；
  3. 双开双合（双掌一线，双肘胸前胸前一线合肘，双膝内合，右掌心斜向上，左掌尖在肩颈窝处）；
  4. 右脚上步划内弧斜撤步开；
  5. 双开双合（双拳一线，双肘胸前一线合肘，双膝内合，左掌心斜向上，右掌尖在肩颈窝处）；
  6. 左脚上步划内弧开，下踏松胯。

第23式 退步压肘

倒卷肱定式

  1. 棚；
  2. 捋靠；

  2. 转向西北合（双臂划圆合，左掌虎口搭右小臂，左掌心向下，右掌心向上）；
  3. 踏掌跟（踏右掌）扭腰转体摆脚（右脚尖尽量向外摆）；
  4. 重心前移，左脚左前方（西北方向）上步成扑步，左腿外侧切掌，右掌向上棚（太阳穴）；
  5. 倒重心转体盖掌（成右扑步，右掌在左腋下贴左肩窝，掌心向外，左掌自左向上向右向下盖掌，掌心向下，指尖朝向右脚尖方向）；
  6. 起身做单摆莲（右脚外摆莲用左手击打右脚背，然后海底翻花，落，西南方滑步捋，左脚滑步成右偏马步），同时双掌变拳捋，左拳到腹前中线，掌心向上，右拳到右胯外侧掌心向下。

第73式 指裆捶

十字单摆莲定式

  1. 双拳上引下打，两掌心向外；
  2. 双拳前棚下捋上打，拳心向内；
  3. 双掌逆缠开 ，双顺缠合，收于腹前，左拳在前，右拳贴左肘，两掌心向上；
  4. 右拳划内弧向下向前崩击，拳心向下，左掌收于腰际，掌心向上。

第74式 白猿献果

指裆捶定式

  1. 左拳不动，右拳逆缠向右上提至太阳穴，左肘前合，眼看肘膝方向；
  2. 摆左脚提右膝献一拳（摆左脚向西南，提右膝，右掌自下而上击打对方下颌）。

第75式 小六封四闭

白猿献果定式

  1. 插右脚，右拳收到左腹前，两掌心向上；
  2. 腰带着转体，双拳腹前逆缠变掌向右按。

第76式 第七单鞭

小六封四闭定式

以左脚掌为轴，

  1. 按，穿掌锁喉（左掌斜向前上至对方喉部，右掌随左掌至左肘，掌心向上）；
  2. 扣掌合肘，（右掌旋腕成勾至左肘，左掌托右肘）；
  3. 给肘勾手（右肘右击后伸右臂勾手），左手下沉至气海；
  4. 甩头提膝插脚（头向左甩，提左膝向左后三分之二插脚），勾手竖直；
  5. 靠（移重心，左肩左挤，左脚尖同时摆一下）；
  6. 略扣右脚，倒掌心，左掌自丹田右胸右肩窝穿，踏掌根，转腰同时左掌略走上弧至左膝上，右勾随转动至右膝上，勾手变圆虚握；
  7. 还原，肘膝相合，下塌外輾。

第77式 雀地龙

第七单鞭定式

  1. 向左棚，双掌变拳略走上弧向右捋；
  2. 重心略前移，两腕相搭左拳左上，右拳在下，拳心相对；
  3. 掌心后移，成扑步，穿左拳至左膝内侧，拳心向上，右拳上提，两拳心遥相对。

第78式 上步七星

雀地龙定式

  1. 裆走下弧移重心，成左式弓蹬步，同时左拳自下向上击打对方下颌，拳心向内，右拳收于右胯外侧，拳心向前，两拳心相对；
  2. 上右脚两拳相搭（右脚上步到左脚前，保持横向距离）重心在左，左腕搭右腕，两掌心向内，左拳在里，右拳在外；
  3. 双拳变掌，身体向左转，双掌向内向上前棚，左掌在外，右掌在内；
  4. 胸腰折叠穿掌，右掌在内，掌背相贴，指尖并拢，重心在左，右脚前脚掌着地。

第79式 退步跨虎

上步七星定式

  1. 撤右脚，右掌不动，转左掌，搭右腕成十字手，方向正南，呈正马步，双掌加棚劲；
  2. 开（逆缠），两掌开至两膝同宽；
  3. 合，后弧穿掌（左脚走后弧，脚掌着地，左掌合右肘，力点在右掌根，重心相对）。

第80式 转身双摆莲

退步跨虎定式

  1. 开（双掌踏掌根右上左下）；
  2. 碾左脚掌，摆右脚，摆脚转体，重心右移，提左膝（像上马一样），身体继续转（朝向北偏东），双掌向右捋，左掌到胸前中线；
  3. 走捋拌（左脚向西北擦脚，双掌向右捋）；
  4. 重心左移，成左式弓蹬步。双掌心向下（三吊腕）；
  5. 走摆莲脚（双掌击打右脚背）；
  6. 右脚撤步向东南方向，双掌变拳自上至左下向右划弧击向左前上方，拳心向内。

第81式 当头炮

转身双摆莲定式

  1. 双拳逆缠开；
  2. 顺缠合（左拳在上，右拳在下，收到右腹前）；
  3. 腰用力带动双拳同时向西北弹击，双拳胸前一趟纵线，同时右脚向前镇一小步（增加击打力度）。

第82式 金刚捣碓

当头炮定式

风扫梅花

  1. 双拳变掌向左前弹出；
  2. 向右捋；
  3. 向左棚，导重心成左式弓蹬步；
  4. 上步搭手（上右脚，脚跟着地，左手搭右小臂）；
  5. 摆脚转体（摆右脚，身体转向正东），东北方向上左脚，切左掌（左腿内侧），右掌上棚；
  6. 倒重心，扫右腿转体，双腿东南西北方向，左掌上托，右掌下按；
  7. 腰带动同时翻掌；
  8. 三点同时到位；
  9. 下沉合，再下沉握拳，提，砸。

第83式 收式

  1. 开（腰和双手同时水平逆时针转，双掌心向上收于腰侧）；
  2. 翻掌下按，身体站直，双臂自然下垂，指尖贴裤逢。
